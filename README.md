# Point of Sale System
<hr>
A Java-based POS system with MySQL integration, designed to automate sales transactions efficiently.

## Features
- Barcode scanning for quick product retrieval.
- Invoice generation for completed transactions.
- Real-time database integration.

## Technologies Used
- Java (JFrame for UI)
- MySQL
